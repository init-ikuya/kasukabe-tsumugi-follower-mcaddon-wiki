# はじめに

## プロジェクト概要

**春日部つむぎ フォロワーアドオン**は、Minecraft Bedrock Edition（統合版）に、VOICEVOXのキャラクター「春日部つむぎ」を冒険の相棒（フォロワー）として追加するアドオンです。

単なる見た目の変更（スキン）に留まらず、VOICEVOXの豊かな音声を活かし、プレイヤーの行動や周囲の環境に応じて「喋り、反応する」高品質なキャラクター体験を提供します。

## 主な機能

- **AIフォロワー** — プレイヤーに追従し、共に戦い、指示に従います
- **ボイスシステム** — VOICEVOXで生成した45種類の音声で状況に応じて喋ります
- **好感度システム** — 一緒に過ごすほど仲良くなれる5段階の関係レベル
- **装備システム** — 剣・斧・防具・盾をフル装備可能
- **戦闘システム** — モンスターと一緒に戦ってくれる頼もしい相棒
- **UIメニュー** — 直感的な操作でつむぎに指示を出せます

## 動作要件

| 項目             | 要件                                |
| ---------------- | ----------------------------------- |
| プラットフォーム | Minecraft Bedrock Edition（統合版） |
| 対応バージョン   | 26.0 以降                           |
| 必要な実験機能   | Beta APIs（ベータAPI）              |
| 対応デバイス     | Windows 10/11, iOS, Android         |

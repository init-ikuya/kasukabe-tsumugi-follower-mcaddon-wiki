# 遊び方ガイド

## Step 1: つむぎを見つける

ワールドにスポーンしたつむぎを探すか、以下のコマンドで召喚します:

```text
/summon tsumugi:follower
```

スポーン直後のつむぎは **野生状態** です。近づくと話しかけてきます:

> 「ん〜？ カレーくれたらついてくよ〜？」

## Step 2: カレーを用意する

つむぎをテイム（仲間に）するには **カレー** が必要です。

サバイバルモードでは、作業台でボウル + ベイクドポテト + ニンジン + ステーキ + 小麦を組み合わせてクラフトできます。

クリエイティブモードでは、以下のコマンドでも入手できます:

```text
/give @s tsumugi:curry
```

## Step 3: テイムする

カレーを手に持って、つむぎに向かって使う（右クリック / タップ）と、つむぎが仲間になります！

> 「カレーありがと〜！ これからよろしくね〜！ ついてくね〜！」

テイム成功率は **100%** なので、カレー1個で確実に仲間にできます。

::: tip
テイム時に好感度が **+20** されるので、最初から「知り合い」レベルでスタートします。
:::

## Step 4: 一緒に冒険する

テイムしたつむぎは自動でプレイヤーについてきます。素手でつむぎに話しかけると **メニュー** が開きます。

### メニューでできること

| ボタン       | 機能                           |
| ------------ | ------------------------------ |
| ついてきて！ | つむぎがプレイヤーについてくる |
| 待ってて     | その場で待機する               |
| 装備を見せて | 装備確認・管理メニューを開く   |

## Step 5: 装備を渡す

つむぎに武器や防具を手に持った状態で使うと、装備させることができます。

装備可能なアイテム:

- **武器**: 木〜ネザライトの剣、木〜ネザライトの斧、トライデント（ピッケルは不可）
- **防具**: ヘルメット、チェストプレート、レギンス、ブーツ
- **盾**: シールド

::: tip
装備には **耐久度** があり、戦闘でダメージを受けると消耗します。**エンチャント** 付きの装備も保持されます。
:::

## Step 6: 一緒に戦う

つむぎは以下の条件で **戦闘モード** に入ります:

- モンスターからダメージを受けた時
- プレイヤーがモンスターからダメージを受けた時（15ブロック以内）

戦闘中はモンスターを自動で攻撃してくれます！周囲10ブロック以内にモンスターがいなくなると自動で戦闘モードが解除されます。

::: warning
体力が20%以下になると **逃走モード** に入ります。回復すると自動で戦闘に復帰します。
:::

## Step 7: 仲良くなる

一緒に過ごすと好感度が上がり、つむぎのセリフが変化します:

| 好感度 | 関係レベル | つむぎの反応                                 |
| ------ | ---------- | -------------------------------------------- |
| 0-19   | 他人       | 「え、なに〜？」                             |
| 20-39  | 知り合い   | 「ん〜？なに〜？」                           |
| 40-59  | 友達       | 「あ、どしたの〜？」                         |
| 60-79  | 親友       | 「最近めっちゃ仲良くなったよね〜！」         |
| 80-100 | 大親友     | 「ねぇねぇ、あーしたち最高の相棒だよね〜！」 |

好感度を上げるには:

- **カレーをあげる**: +10
- **近くにいる**（5ブロック以内、60秒ごと）: +1

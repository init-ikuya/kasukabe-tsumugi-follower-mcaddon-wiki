# インストール方法

## ダウンロード

[CurseForge のアドオンページ](https://www.curseforge.com/minecraft-bedrock/addons/kasukabe-tsumugi-follower)から最新の `.mcaddon` ファイルをダウンロードします。

## インストール手順

### Windows

1. ダウンロードした `.mcaddon` ファイルをダブルクリック
2. Minecraft が自動で起動し、アドオンがインポートされます
3. ワールド設定でアドオンを有効化（下記参照）

### iOS / Android

1. ブラウザから `.mcaddon` ファイルをダウンロード
2. ファイルを開く → 「Minecraft で開く」を選択
3. 自動でインポートされます

## ワールド設定

### 新規ワールドの場合

1. 「遊ぶ」→「新しく作る」→「世界を新しく作る」
2. 左メニューの「アドオン」を選択
3. 「ビヘイビアーパック」タブで **春日部つむぎ フォロワー** を有効化
4. 「リソースパック」タブで対応するリソースパックも有効化されていることを確認
5. 以下の実験機能を有効にします:

::: warning 必須設定
**ベータAPI**（Beta APIs）を必ず有効にしてください。この設定がないとアドオンが正しく動作しません。
:::

### 既存ワールドの場合

1. ワールド一覧で対象ワールドの右の鉛筆アイコンをタップ
2. 「アドオン」から同様にパックを有効化
3. 「実験」から **ベータAPI** を有効化

## インストール確認

ワールドに入ったら、以下のコマンドでつむぎをスポーンできます:

```text
/summon tsumugi:follower
```

つむぎが出現すれば、インストール成功です！

## トラブルシューティング

| 症状                   | 対処法                                         |
| ---------------------- | ---------------------------------------------- |
| つむぎがスポーンしない | ベータAPIが有効か確認してください              |
| 音声が再生されない     | リソースパックが有効か確認してください         |
| メニューが開かない     | ベータAPIが有効か確認してください              |
| `.mcaddon` が開けない  | Minecraft のバージョンを最新に更新してください |

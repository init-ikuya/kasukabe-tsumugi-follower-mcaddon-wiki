# よくある質問

## 一般的な質問

### つむぎが動かない / 反応しない

ワールド設定で **ベータAPI**（Beta APIs）が有効になっているか確認してください。この設定が無効だとつむぎのAI機能が使えません。

### つむぎが喋らない

リソースパックが正しく有効化されているか確認してください。音声ファイルはリソースパック側に含まれています。

### テイムできない

野生のつむぎに **カレー**（`tsumugi:curry`）を使う必要があります。他のアイテムではテイムできません。

サバイバルモードでは、作業台でボウル + ベイクドポテト + ニンジン + ステーキ + 小麦を組み合わせてクラフトできます（ボウルを入手するとレシピがアンロックされます）。

クリエイティブモードでは `/give @s tsumugi:curry` でカレーを入手できます。

### つむぎが消えてしまった

つむぎはプレイヤーから20ブロック以上離れると自動テレポートしますが、もし見つからない場合は以下を試してください:

```text
/tp @e[type=tsumugi:follower] @s
```

## 戦闘について

### つむぎが戦わない

つむぎはデフォルトでは **平和モード** です。以下の条件で戦闘モードに入ります:

- つむぎがモンスターからダメージを受けた
- プレイヤーがモンスターからダメージを受けた（15ブロック以内）

プレイヤーが直接つむぎを殴っても戦闘モードには入りません。モンスターを倒した後は、周囲10ブロック以内にモンスターがいなくなると自動で戦闘モードが解除されます。

### つむぎが逃げてしまう

体力が最大HPの20%以下（8以下）になると **逃走モード** に入ります。体力が25%以上に回復すると自動で戦闘に復帰します。

## 装備について

### 装備を外したい

メニューの「装備を見せて」から装備管理画面を開き、外したい装備のボタンを押してください。外した装備はアイテムとしてドロップされます。

### つむぎが死んだら装備はどうなる？

つむぎが死亡すると **装備は死亡位置にドロップ** されます。エンチャント情報も保持された状態でドロップされます。つむぎ自体は10秒後に自動リスポーンし、テイム状態と好感度は引き継がれます（死亡ペナルティとして好感度 -5）。

### 装備が壊れた

装備には耐久度があり、戦闘でダメージを受けると消耗します。耐久度が0になると装備は壊れて消滅します。装備確認メニューから残り耐久度を確認できます。

## その他

### 複数体テイムできる？

はい、複数体のつむぎをテイムできます。それぞれ独立した好感度・装備を持ちます。

### マルチプレイで使える？

はい、マルチプレイに対応しています。各プレイヤーがそれぞれのつむぎをテイムできます。

### 他のアドオンと一緒に使える？

基本的に他のアドオンと併用可能ですが、一部のアドオンとは競合する可能性があります。

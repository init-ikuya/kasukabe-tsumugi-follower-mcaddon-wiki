# コマンド一覧

ゲーム内で使えるコマンドの一覧です。チャット欄に入力して使用します。

## つむぎを召喚する

```text
/summon tsumugi:follower
```

つむぎを現在の位置にスポーンさせます。スポーン直後は野生状態です。

## カレーを入手する

```text
/give @s tsumugi:curry
```

テイムや回復に使えるカレーを入手します。

## つむぎをテレポートさせる

```text
/tp @e[type=tsumugi:follower] @s
```

全てのつむぎを自分の位置にテレポートさせます。つむぎが見つからない時に便利です。

## つむぎを削除する

```text
/kill @e[type=tsumugi:follower]
```

全てのつむぎを削除します。テイム済みの場合、10秒後に自動リスポーンします。

# 変更履歴

## v1.0.0

初回リリース

### 機能

- テイムシステム（カレーによる100%テイム）
- 好感度システム（5段階の関係レベル）
- 戦闘システム（戦闘モード・逃走モード）
- 装備システム（剣・斧・防具・盾対応）
- 追従・ナビゲーション（自動テレポート・ディメンション移動対応）
- ボイスシステム（VOICEVOX 45種類の音声）
- UIメニュー（ステータス確認・指示・装備管理）
- 乗り物対応（ボート・イカダ・トロッコ）
- 時間帯セリフ（朝・夕・夜）
- 雷反応セリフ
- リスポーンシステム（10秒後自動復活・状態引き継ぎ）

# 変更履歴

## v1.0.0

正式リリース。Minecraft 26.0 対応、装備システムの大幅強化、戦闘システムの改善など。

### 動作要件の変更

- 対応Minecraftバージョンを **26.0 以降** に変更

### 新機能

- **装備耐久度システム**: 武器・防具に耐久値が設定され、戦闘でダメージを受けると消耗する
- **エンチャント保持**: エンチャント付き装備を渡すと情報が保持され、外した時・死亡時にも引き継がれる
- **装備確認UIに耐久度表示**: パーセントとバーで残り耐久度を確認可能
- **戦闘モード自動終了**: 周囲10ブロック以内にモンスターがいなくなると自動で戦闘モード解除
- **野生つむぎの行動制限**: ベッドの近くにスポーンし、半径30ブロック以内で行動
- **カレーのクラフトレシピ**: ボウル + ベイクドポテト + ニンジン + ステーキ + 小麦で作成可能
- **装備可能アイテムの表示**: 装備可能なアイテムを持っている時「装備させる」と表示

### 改善

- カレー使用後にボウルがプレイヤーに返却されるように
- 装備を外した時にアイテムが地面にドロップされるように
- 待機中でも好感度が上がるように
- 音声の聴取距離外ではテキストメッセージが表示されないように
- 音声の音量を1.0から1.3に増加
- 野生つむぎが水に入らないように
- 水中の移動速度を改善

### バグ修正

- サバイバルモードでテイムできない問題を修正
- 戦闘モードがモンスター撃破後も終了しない問題を修正
- ピッケルが武器として装備できてしまう問題を修正

### ボイス追加

- 装備時: 「わ〜い！{item}装備したよ〜！」
- 装備解除時: 「はーい、{item}外したよ〜！」
- スロット使用中: 「には{item}があるよ〜先に外してね〜」
- 装備破損時: 「あっ! {item}壊れちゃった〜!」

## v1.0.0-beta.5 以前

### 機能

- テイムシステム（カレーによる100%テイム）
- 好感度システム（5段階の関係レベル）
- 戦闘システム（戦闘モード・逃走モード）
- 装備システム（剣・斧・防具・盾対応）
- 追従・ナビゲーション（自動テレポート・ディメンション移動対応）
- ボイスシステム（VOICEVOX 45種類の音声）
- UIメニュー（ステータス確認・指示・装備管理）
- 乗り物対応（ボート・イカダ・トロッコ）
- 時間帯セリフ（朝・夕・夜）
- 雷反応セリフ
- リスポーンシステム（10秒後自動復活・状態引き継ぎ）

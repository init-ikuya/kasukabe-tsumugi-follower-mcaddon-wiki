# カレーアイテム

## 概要

カレーはつむぎアドオン専用のアイテムで、テイム・回復・好感度アップの3つの機能を持つ万能アイテムです。

## 入手方法

### クラフトで作る

作業台で以下の材料を組み合わせるとカレーが作れます（不定形レシピ）:

| 材料                     |
| ------------------------ |
| ボウル × 1               |
| ベイクドポテト × 1       |
| ニンジン × 1             |
| ステーキ（焼き牛肉） × 1 |
| 小麦 × 1                 |

::: tip レシピのアンロック
**ボウル** を手に入れるとレシピが自動でアンロックされます。
:::

### コマンドで入手

クリエイティブモードでは以下のコマンドでも入手できます:

```text
/give @s tsumugi:curry
```

## カレーの効果

### 野生のつむぎに使った場合

- **テイム成功**（100%の確率）
- 好感度 +20（テイムボーナス）

### テイム済みのつむぎに使った場合

- **体力回復**: 10ポイント回復
- **好感度アップ**: +10
- 使用後 **ボウル** がプレイヤーに返却されます

## 数値詳細

| 項目           | 値  |
| -------------- | --- |
| 体力回復量     | 10  |
| 好感度ボーナス | +10 |
| テイム時好感度 | +20 |

::: tip
つむぎの最大HPは40なので、カレー4個で全回復できます。戦闘後のケアにも使えます！
:::

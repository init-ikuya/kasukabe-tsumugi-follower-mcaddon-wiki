# 装備システム

## 概要

つむぎにアイテムを渡すことで武器・防具を装備させることができます。装備した武器・防具は見た目にも反映されます。

## 装備スロット

| スロット | 日本語名 | 装備可能アイテム |
|---------|---------|----------------|
| mainhand | メイン武器 | 剣、斧、トライデント |
| offhand | 盾 | シールド |
| head | 頭 | ヘルメット |
| chest | 胴体 | チェストプレート |
| legs | 脚 | レギンス |
| feet | 足 | ブーツ |

## 装備の操作

### 装備する

1. 装備させたいアイテムを手に持つ
2. つむぎに向かって使う（右クリック / タップ）

### 装備を外す

1. メニュー → 「装備を見せて」
2. 外したい部位のボタンを押す
3. アイテムがドロップされる

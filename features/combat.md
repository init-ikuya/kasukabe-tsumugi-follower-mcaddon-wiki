# 戦闘システム

## 概要

つむぎはテイム後、条件を満たすと戦闘モードに入り、モンスターと戦ってくれます。

## 戦闘モード

### 突入条件

- つむぎがモンスターからダメージを受けた時
- プレイヤーがモンスターからダメージを受けた時（15ブロック以内）

::: info
プレイヤーが直接つむぎを殴っても戦闘モードには入りません。
:::

### 戦闘行動

| 項目 | 値 |
|------|-----|
| 攻撃対象 | モンスター系のエンティティ |
| 攻撃範囲 | 16ブロック |
| 攻撃速度倍率 | 1.1倍 |
| 基本攻撃力 | 2 |
| 武器装備時 | 武器のダメージ値が適用 |

### 戦闘モードの自動終了

つむぎの周囲 **10ブロック以内** にモンスターがいなくなると、自動的に戦闘モードが解除されます。チェックは10秒ごとに行われます。

### セリフ

- 戦闘モード突入時: 「おっけー！ 戦うよ〜！ 任せて〜！」
- 戦闘モード解除時: 「りょーかい！ 戦うのやめるね〜」

## 逃走モード

体力が低下すると自動で戦闘を離脱し、逃走します。

| 項目 | 値 |
|------|-----|
| 発動条件 | 体力が最大HPの20%以下（HP 8以下） |
| 逃走速度 | 通常の1.5〜1.8倍 |
| 復帰条件 | 体力が25%以上（HP 10以上）に回復 |

### セリフ

- 逃走開始時: 「やばっ...！ 逃げるね...！」
- 復帰時: 「回復したよ〜！ また戦えるね！」

## ダメージ関連セリフ

| 状況 | セリフ |
|------|--------|
| モンスターから攻撃を受けた | 「いったぁ！ やるね〜！」 |
| オーナーに殴られた | 「え...なんで殴るの...？」 |
| オーナーが攻撃を受けた | 「大丈夫！？ 許さないんだから！」 |

## 平和モード

戦闘モードではない通常の状態です。モンスターに攻撃されるまで平和モードを維持します。

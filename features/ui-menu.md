# UIメニュー

## 概要

テイム済みのつむぎに素手でインタラクトすると表示されるメニューです。つむぎのステータス確認や指示を出すことができます。

## メインメニュー

### ステータス表示

メニュー上部にはつむぎの現在のステータスが表示されます:

- **タイトル**: 春日部つむぎ + 状態表示
- **体調**: 体力に応じた4段階表示
- **気持ち**: 好感度に応じた表示
- **関係**: 現在の関係レベル

#### 状態表示

| 状態 | 表示 |
|------|------|
| 追従中 | （ついてきてる） |
| 待機中 | （待機中） |
| 戦闘中 | 【戦闘中】 |

#### 体調表示

| 体力割合 | 表示 |
|---------|------|
| 高い | 元気いっぱい！ |
| やや高い | まぁまぁかな〜 |
| やや低い | ちょっと疲れてきた… |
| 低い | けっこうキツイかも… |

#### 気持ち表示

| 好感度レベル | 表示 |
|-------------|------|
| 大親友 | 大好き！ |
| 親友 | 仲良し〜♪ |
| 友達 | いい感じかも？ |
| 知り合い | まだよくわかんないけど… |
| 他人 | …誰？ |

### ボタン

| ボタン | 機能 |
|--------|------|
| ついてきて！ | つむぎがプレイヤーについてくる |
| 待ってて | その場で待機する |
| 装備を見せて | 装備確認メニューを開く |

## 装備確認メニュー

「装備を見せて」を押すと装備管理画面が開きます。

### 表示内容

- **現在の装備一覧**: 各スロットの装備状況
- **武器ダメージ**: 装備中の武器のダメージ値
- **防御力**: 装備中の防具の合計防御値

### ボタン

| ボタン | 機能 |
|--------|------|
| 武器を外す | 武器を外す |
| 盾を外す | 盾を外す |
| 防具を全部外す | 全防具を一括で外す |
| 戻る | メインメニューに戻る |

## 装備スロット

| スロット | 対応部位 |
|---------|---------|
| メイン武器 | 武器 |
| 盾 | オフハンド |
| 頭 | ヘルメット |
| 胴体 | チェストプレート |
| 脚 | レギンス |
| 足 | ブーツ |

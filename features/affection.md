# 好感度システム

## 概要

つむぎとの親密度を表すシステムです。好感度が上がるとセリフや反応が変化します。

## 好感度の上げ方

| アクション     | 好感度変化 | 備考                                    |
| -------------- | ---------- | --------------------------------------- |
| テイム時       | +20        | 初回のみ                                |
| カレーをあげる | +10        | 何度でも可                              |
| 近くにいる     | +1         | 5ブロック以内、60秒ごと（待機中も有効） |
| つむぎが死亡   | -5         | 死亡ペナルティ                          |

## 好感度レベル

好感度は最大 **100** で、5段階の関係レベルがあります:

| 好感度 | 関係レベル |
| ------ | ---------- |
| 0-19   | 他人       |
| 20-39  | 知り合い   |
| 40-59  | 友達       |
| 60-79  | 親友       |
| 80-100 | 大親友     |

## レベル別セリフ

### 大親友（80-100）

- 「ねぇねぇ、あーしたち最高の相棒だよね〜！」
- 「ずっと一緒にいようね〜！ 大好き〜！」

気持ち表示: 大好き！

### 親友（60-79）

- 「最近めっちゃ仲良くなったよね〜！」
- 「一緒にいると楽しいな〜！」

気持ち表示: 仲良し〜♪

### 友達（40-59）

- 「あ、どしたの〜？」
- 「なんか用〜？」

気持ち表示: いい感じかも？

### 知り合い（20-39）

- 「ん〜？なに〜？」
- 「カレーくれたらもっと仲良くなれるかも〜？」

気持ち表示: まだよくわかんないけど…

### 他人（0-19）

- 「え、なに〜？」
- 「カレーくれたら話聞くよ〜？」

気持ち表示: …誰？

## UIでの表示

メニュー画面では以下の情報が表示されます:

- **体調**: 体力に応じた4段階表示
- **気持ち**: 好感度に応じた表示
- **関係**: 現在の関係レベル

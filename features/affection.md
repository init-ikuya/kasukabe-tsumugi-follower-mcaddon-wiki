# 好感度システム

## 概要

つむぎとの親密度を表すシステムです。好感度が上がるとセリフや反応が変化します。

## 好感度の上げ方

| アクション     | 好感度変化 | 備考                                    |
| -------------- | ---------- | --------------------------------------- |
| テイム時       | +20        | 初回のみ                                |
| カレーをあげる | +10        | 何度でも可                              |
| 近くにいる     | +1         | 5ブロック以内、60秒ごと（待機中も有効） |
| つむぎが死亡   | -5         | 死亡ペナルティ                          |

## 好感度レベル

好感度は最大 **100** で、5段階の関係レベルがあります:

| 好感度 | 関係レベル |
| ------ | ---------- |
| 0-19   | 他人       |
| 20-39  | 知り合い   |
| 40-59  | 友達       |
| 60-79  | 親友       |
| 80-100 | 大親友     |

## レベル別セリフ

### 大親友（80-100）

- 「ねぇねぇ、あーしたち最高の相棒だよね〜！」
- 「ずっと一緒にいようね〜！ 大好き〜！」

気持ち表示: 大好き！

### 親友（60-79）

- 「最近めっちゃ仲良くなったよね〜！」
- 「一緒にいると楽しいな〜！」

気持ち表示: 仲良し〜♪

### 友達（40-59）

- 「あ、どしたの〜？」
- 「なんか用〜？」

気持ち表示: いい感じかも？

### 知り合い（20-39）

- 「ん〜？なに〜？」
- 「カレーくれたらもっと仲良くなれるかも〜？」

気持ち表示: まだよくわかんないけど…

### 他人（0-19）

- 「え、なに〜？」
- 「カレーくれたら話聞くよ〜？」

気持ち表示: …誰？

## 好感度ステータスバフ {#stat-buffs}

::: info v1.1.0 で追加
:::

好感度レベルが上がると、つむぎの戦闘能力が自動的に強化されます。

### 攻撃ボーナス

| 好感度レベル | 攻撃力ボーナス |
| ------------ | -------------- |
| 友達（40〜）  | +1            |
| 大親友（80〜）| +2            |

武器のダメージに加算されます。素手の場合は基本攻撃力（2）に加算されます。

### ダメージ軽減

| 好感度レベル    | ダメージ軽減 |
| --------------- | ------------ |
| 友達（40〜）     | 5%          |
| 親友（60〜）     | 10%         |
| 大親友（80〜）   | 15%         |

防具の防御力に加えて、好感度による軽減が適用されます。防具を装備していなくても効果があります。

### HP自動回復

| 好感度レベル    | 回復間隔 |
| --------------- | -------- |
| 知り合い以下    | 5秒ごと  |
| 友達（40〜）     | 4秒ごと  |
| 大親友（80〜）   | 3秒ごと  |

1回につき1HP回復します。戦闘中でも回復します。

## UIでの表示

メニュー画面では以下の情報が表示されます:

- **体調**: 体力に応じた4段階表示
- **気持ち**: 好感度に応じた表示
- **関係**: 現在の関係レベル

# ボイスシステム

## 概要

つむぎは VOICEVOXで生成された **45種類** のボイスを使い、状況に応じてリアルに喋ります。

## ボイスの種類

### 時間帯セリフ

ゲーム内の時間帯が変わるとつむぎが話しかけてきます（16ブロック以内のみ）。

| 時間帯 | Minecraft時間 | セリフ例 |
|--------|-------------|---------|
| 朝 | 0〜 | 「おはよ〜！ 今日も一日がんばろ〜！」 |
| 夕方 | 12000〜 | 「そろそろ暗くなってきたね〜」 |
| 夜 | 13000〜 | 「夜だ〜！ モンスター気をつけてね〜！」 |

### 好感度セリフ

つむぎに話しかけた時、好感度レベルに応じたセリフが再生されます。

| レベル | セリフ例 |
|--------|---------|
| 大親友 | 「ねぇねぇ、あーしたち最高の相棒だよね〜！」 |
| 親友 | 「最近めっちゃ仲良くなったよね〜！」 |
| 友達 | 「あ、どしたの〜？」 |
| 知り合い | 「ん〜？なに〜？」 |
| 他人 | 「え、なに〜？」 |

### 戦闘セリフ

| 状況 | セリフ |
|------|--------|
| 戦闘開始 | 「おっけー！ 戦うよ〜！ 任せて〜！」 |
| 戦闘終了 | 「りょーかい！ 戦うのやめるね〜」 |
| 被ダメージ | 「いったぁ！ やるね〜！」 |
| オーナー攻撃検知 | 「大丈夫！？ 許さないんだから！」 |
| 逃走開始 | 「やばっ...！ 逃げるね...！」 |
| 逃走復帰 | 「回復したよ〜！ また戦えるね！」 |

### 装備セリフ

| 状況 | セリフ |
|------|--------|
| 装備成功 | 「わ〜い！{item}装備したよ〜！」 |
| 装備解除 | 「はーい、{item}外したよ〜！」 |
| 装備破損 | 「あっ! {item}壊れちゃった〜!」 |
| 装備不可 | 「ん〜？これ装備できないかも〜？」 |
| スロット使用中 | 「には{item}があるよ〜先に外してね〜」 |

### テイム・カレーセリフ

| 状況 | セリフ |
|------|--------|
| テイム成功 | 「カレーありがと〜！ これからよろしくね〜！ ついてくね〜！」 |
| カレー受け取り | 「カレーありがと〜！ 大好き〜！」 |
| 既にテイム済み | 「もうテイムされてるよ〜？」 |

### その他セリフ

| 状況 | セリフ例 |
|------|---------|
| 雷 | 「きゃっ！ びっくりした〜！」 |
| 乗り物 | 「一緒に乗るね〜！」 |
| 野生時 | 「ん〜？ カレーくれたらついてくよ〜？」 |
| リスポーン | 「ふぅ〜...戻ってきたよ〜！ ちょっと痛かった〜...」 |
| オーナー復帰 | 「おかえり〜！ 待ってたよ〜！」 |

## 音声の仕様

| 項目 | 値 |
|------|-----|
| 生成エンジン | VOICEVOX |
| キャラクター | 春日部つむぎ |
| 聴取距離 | 16ブロック |
| 総ボイス数 | 45種類 |

::: tip 全ボイスラインの一覧
全45種類のセリフは[ボイスライン一覧](/reference/voice-lines)で確認できます。
:::

# 日記システム

::: info v1.1.0 で追加
この機能は v1.1.0 で追加されました。
:::

## 概要

つむぎとの冒険の記録が詰まった日記アイテムです。好感度が「大親友」になるとメニューからもらえるようになります。

## 入手方法

1. 好感度を **80以上**（大親友）まで上げる
2. メニューを開くと「**日記をもらう**」ボタンが表示される
3. ボタンを押すと日記がインベントリに追加される

> 「あーしの日記、あげるね〜！ 大事にしてね〜！」

::: tip
日記は何度でももらい直すことができます。最新のデータに更新された日記が手に入ります。
:::

## 日記の内容

### ステータスページ（1ページ目）

日記の最初のページには、つむぎとの関係の概要が表示されます:

| 項目         | 内容                 |
| ------------ | -------------------- |
| 関係         | 現在の関係レベル     |
| 好感度       | ハートゲージで表示   |
| 一緒にいた時間 | 合計時間           |
| 討伐数       | 倒したモンスターの数 |
| 出会った日   | テイムした日付       |

### 日記エントリ（2ページ目以降）

2ページ目以降には、日ごとの冒険の記録がつむぎの言葉で書かれています。

各日のエントリには以下の情報が含まれます:

- **カレーの回数**: その日にカレーをもらった回数
- **討伐記録**: 倒したモンスターの種類と数
- **やられた回数**: つむぎが倒された回数
- **一緒にいた時間**: その日に一緒に過ごした時間

何もイベントがなかった日には、つむぎのひとりごとが書かれます:

> 「今日はいい天気だったな〜」「明日はカレー食べたいな〜」

::: info
日記には最大30日分の記録が保存されます。
:::

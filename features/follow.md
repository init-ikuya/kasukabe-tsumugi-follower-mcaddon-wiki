# 追従・ナビゲーション

## 概要

テイム済みのつむぎはプレイヤーを自動で追従します。距離に応じた行動と、ディメンション間のテレポートにも対応しています。

## 追従距離

| 距離 | 行動 |
|------|------|
| 5ブロック以内 | 停止（プレイヤーの近くで待機） |
| 5〜10ブロック | 移動なし |
| 10ブロック以上 | 追従開始（プレイヤーに向かって移動） |
| 20ブロック以上 | 自動テレポート |

## テレポート

以下の条件でつむぎはプレイヤーの近くにテレポートします:

- プレイヤーとの距離が **20ブロック以上** 離れた場合
- プレイヤーと **異なるディメンション** にいる場合

::: info
テレポート先はプレイヤーの近くのランダムな位置です。
:::

## 乗り物

つむぎはプレイヤーが乗り物に乗った時、一緒に乗ることができます。

### 対応する乗り物

- ボート（全木材タイプ）
- チェストボート
- 竹イカダ / チェスト竹イカダ
- トロッコ

### セリフ

- 「一緒に乗るね〜！」
- 「出発進行〜！」

## 待機モード

メニューから「待ってて」を選ぶと、つむぎはその場で待機します。

- 待機中はプレイヤーを追従しません
- 「ついてきて！」で追従を再開できます

### セリフ

| 状況 | セリフ |
|------|--------|
| 追従開始 | 「おっけー！ ついてくね〜！」 |
| 待機開始 | 「りょーかい！ ここで待ってるね〜」 |

## 水中での挙動

つむぎが水中で溺れダメージを受けると **浮上モード** に入り、水面を目指します。水から出ると通常の追従状態に戻ります。
